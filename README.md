# Animated React Portfolio

## Run locally

```bash
npm install
npm run dev
```

Edit your name, links, projects, and photo path in `src/content.js`.

## Add your photo

1. Put your photo at `public/me.jpg`.
2. In `src/content.js`, set `photo: '/me.jpg'`.

## Deploy to Netlify

Push the project to GitHub, import it in Netlify, and deploy. The included `netlify.toml` configures the build automatically.
